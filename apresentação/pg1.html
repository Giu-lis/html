CSS

Cascading Style Sheet ou Folhas de Estilo em Cascata.

O CSS é uma linguagem de estilo, utilizado para definir como vai ser a aparência de um site em conjunto com linguagens de marcação. Ele é como a decoração de uma casa! Enquanto o HTML é a estrutura e os móveis, o CSS é a decoração.

O principal objetivo do CSS é separar o design do conteúdo da linguagem de marcação na hora da escrita. Esse foi o motivo do CSS ter sido criado, para facilitar a definição da aparência do site, pois permite uma maior flexibilidade e controle do código.
